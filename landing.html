GitHub Actions

Automate it. Set and forget.

The weekly schedule is intentional â€” context rot happens even when AGENTS.md hasn't changed.

# .github/workflows/agents-lint.yml
name: agents-lint
on:
  push:
    paths: ['AGENTS.md', 'package.json']
  schedule:
    - cron: '0 9 * * 1' # Weekly Monday

jobs:
  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: npx agents-lint --max-warnings 5
